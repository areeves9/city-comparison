# City Comparison — Job Market Intelligence Dashboard

A React dashboard comparing US tech job markets across employment, compensation, cost of living, and profile-fit metrics for five cities: Portland, Austin, Raleigh-Durham, Seattle, and Dallas-Fort Worth.

## Features

- Sortable city cards by fit score, unemployment rate, or name
- Expandable detail panels with employment, tech market, compensation, and profile-fit metrics
- Color-coded indicators for growing vs. contracting markets
- Key findings with contextual highlighting
- Dark theme with monospace typography

## Data Sources

BLS, Indeed, LinkedIn, Colliers, ULI, Portland Metro Chamber, NC Tech Association

## Tech Stack

- React 18
- Vite
- Hosted as a static site on Render

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Output goes to `dist/`.
